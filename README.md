# NodeNote
 :notebook: Take Notes on your Terminal! 
### Description 
Quickly take notes, list them, fetch them, delete them, all from the terminal. 

### Usage
>Add a note 

`npm run add 'title' -- --body 'text_note'`

>Read a note

`npm run read 'title_name'`

>List all notes

`npm run list'`

>Delete a note

`npm run delete 'title_name'`
